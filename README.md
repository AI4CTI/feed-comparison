# feed-comparison

A reproducible command-line tool to compare and benchmark feeds of malicious URLs.

> **Status — early access.** This is the initial public release (`v0.1.0`) of a tool originally developed inside Ermes Browser Security. The first public version exposes a curated subset of the original feature set; see [CHANGELOG.md](CHANGELOG.md) for what is included and what is intentionally not yet shipped.

## About

`feed-comparison` lets researchers and analysts download samples from several threat-intelligence feeds, normalise the URLs they expose, and quantify how those feeds overlap and how their discovery times compare. It is developed within [AI4CTI](https://ai4cti.polito.it/), a joint research initiative between Politecnico di Torino and Ermes Browser Security on AI-driven Cyber Threat Intelligence.
